# ERP-Nav (Phase 1 MVP)

ERP-Nav is a neutral, AI-powered ERP marketplace concept focused on helping buyers learn, assess readiness, compare products, and connect with implementation partners.

This repository contains a **Phase 1 MVP starter** with:

- ERP 101 and vendor explainers entry content
- Email-gated ERP Readiness Assessment
- Basic TCO and ROI calculators
- Feature parity matrix for 10 vendors/OEMs
- Partner directory with lead capture form
- Simple downloadable RFP generator

## Included Vendors/OEMs

- Oracle
- SAP
- Microsoft
- Infor
- Epicor
- Workday
- Acumatica
- Sage
- Odoo
- IFS

## Run locally

No build step required.

1. Open `index.html` directly in a browser, **or**
2. Serve the folder:

```bash
python3 -m http.server 8080
```

Then open <http://localhost:8080>.

## Phase 1 scope implemented

- Readiness assessment and lead capture form
- Template-based TCO + ROI estimate blocks

- RFP builder + downloadable text output
